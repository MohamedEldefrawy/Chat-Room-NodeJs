## My process

1) Clone the project

   ``` https://github.com/MohamedEldefrawy/Chat-Room-NodeJs.git```
2) install [node](https://nodejs.org/en/download/)
3) in the project directory run the following
4) ```
   npm install
   ```
    ```
    npm run dev
    ```
